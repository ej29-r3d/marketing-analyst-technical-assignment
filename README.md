# Technical Assignments Repository

This repository contains technical assignments for different roles at Improvado.

## Available Assignments

### 1. [Marketing Analyst Assignment](./marketing-analyst-assignment/)
Technical assignment for Senior Marketing Analyst candidates focusing on multi-channel data unification and dashboard creation.

**Key Requirements:**
- Database setup and data integration
- Dashboard visualization
- SQL transformation scripts

### 2. [Engagement Lead Assignment](./engagement-lead-assignment/)
Technical assignment for Engagement Lead candidates emphasizing cross-channel performance analysis and client presentation skills.

**Key Requirements:**
- Database setup and data unification
- Dashboard creation
- Video walkthrough (technical + client presentation)

### 3. [Technical Customer Success Assignment](./tcs-assignment/)
Dashboard recreation challenge for Technical Customer Success candidates focusing on visualization skills and insight communication.

**Key Requirements:**
- Dashboard recreation from screenshot
- Dummy data generation
- Video walkthrough explaining dashboard usage and insights

## Data Files

Both assignments use the same advertising data from three major platforms:
- `01_facebook_ads.csv` - Facebook advertising metrics
- `02_google_ads.csv` - Google Ads campaign data
- `03_tiktok_ads.csv` - TikTok advertising engagement
